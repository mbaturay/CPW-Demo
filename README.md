# CPW Demo

Environmental data dashboard demo built with React, Vite, and Tailwind CSS.

[![React](https://img.shields.io/badge/React-18.3.1-61DAFB?logo=react&logoColor=000000)](https://react.dev/)
[![Vite](https://img.shields.io/badge/Vite-6.3.5-646CFF?logo=vite&logoColor=ffffff)](https://vitejs.dev/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4.1.12-38B2AC?logo=tailwindcss&logoColor=ffffff)](https://tailwindcss.com/)

## Setup

- Install dependencies: `npm install`
- Start the dev server: `npm run dev`
- Build for production: `npm run build`

## Notes

- The app runs at http://localhost:5173/ by default.
